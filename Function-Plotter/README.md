# Function Plotter

---

## How to use

### Installation

1. Clone the repository:
   ```bash
   git clone https://git.thm.de/nobk36/function-plotter
   ```
   
2. Navigate to the project directory:
   ```bash
    cd function-plotter
    ```

### Run the application

1. Run via gradle:
   ```bash
   ./gradlew run
   ```
   
## Testing

### Run tests

1. To run all tests, use:
   ```bash
   ./gradlew test
   ```
